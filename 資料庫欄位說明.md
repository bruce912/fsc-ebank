## 通用欄位

以下欄位出現於大多數資料表，定義一致：

| 欄位名稱 | 型別 | 說明 |
|---|---|---|
| `yr` | INTEGER | 年份（民國年，例：115 表示民國115年） |
| `mn` | INTEGER | 月份（1–12） |
| `ym` | TEXT | 年月組合（格式：`YY/MM`，例：`115/02`） |
| `機構名稱` | TEXT | 申報機構全名 |
| `資料月份` | TEXT | 資料月份（中文格式，例：`115 年 02 月`） |
| `維護部門名稱` | TEXT | 申報機構負責填報之部門名稱 |
| `主管姓名` | TEXT | 維護部門主管姓名 |
| `主管電話` | TEXT | 維護部門主管電話 |
| `承辦人姓名` | TEXT | 實際填報承辦人員姓名 |
| `承辦人電話` | TEXT | 承辦人員電話 |
| `承辦人E_MAIL` | TEXT | 承辦人員電子郵件 |

> **關於系統代碼欄位**：各表中出現的 `c1`、`c2`、`c3`、`c4`、`c5`、`c6`、`s1`–`s4`、`sn`、`bank_no` 等短代碼欄位，為 FSC 申報系統的內部分類維度代碼（整數索引），代表各資料列所對應的分組條件（如業務別、帳戶類別、通路別等）。`_N__amt`、`_N__qty`、`_N__cards`、`_N__xact`、`_N__remark` 等格式欄位為系統動態展開的第 N 個數值欄位，與同表的中文命名欄位對應相同業務數據。

### EC002W 儲值卡發行資料維護

每列代表一家機構、一張儲值卡、一個月的資料。報表為指標列表形式，每個指標有兩欄：**全部（含重複加值式）** 與 **重複加值式**。

**報表結構：**

```
列方向（指標項目）                    欄方向
                              全部(含重複加值式)  重複加值式
發卡總數（累計至本月底）              ●              ●
流通卡數（發卡總數扣除停退卡）         ●              ●
當月發卡數                           ●              ●
當月停卡數                           ●              ●
當月贖回金額（元）                    ●              ●
當月偽冒停卡張數                      ●              ●
當年度累計偽冒停卡張數                ●              ●
當月偽冒損失金額（元）                ●              ●
當年度累計偽冒損失金額（元）           ●              ●
```

**欄位對應說明：**

| 欄位名稱 | 型別 | 中文定義 |

| `yr` | INTEGER | 年份（民國年） |
| `mn` | INTEGER | 月份 |
| `ym` | TEXT | 年月（YY/MM） |

### EC011W 儲值卡類型資料維護

每列代表一家機構一個月的完整儲值卡類型統計。每個卡種對應 5 個欄位（發卡總數、流通卡總數、當月交易筆數、當月交易金額、附註說明），共 18 種卡種（索引 0–17）。

**報表列方向（儲值卡類型，索引 0–17）：**

```
記名
  _0  普通卡
  聯名卡
    _1  與金融機構合作
    _2  與政府機關合作
    _3  與電信業者合作
    _4  與教育部認可之學校合作
    _5  其他合作（公司行號、社區等）
  NFC 手機儲值卡（不含聯名卡）
    _6  中華電信
    _7  台灣大哥大
    _8  遠傳電信
    _9  亞太電信
    _10 其他
  優待卡（不含聯名卡）
    _11 符合相關法令得購買半票之兒童
    _12 年滿65歲以上之民眾
    _13 領有身心障礙手冊人士適用
    _14 其他
  _15 附隨電子支付帳戶儲值卡
無記名
  _16 普通卡（重覆加值式）
  _17 拋棄式儲值卡（指定期間內限次或不限數搭乘）

欄方向：發卡總數 / 流通卡總數 / 當月交易筆數 / 當月交易金額（元）/ 附註說明
```

**欄位對應說明：**

| 欄位名稱 | 型別 | 中文定義 |
|---|---|---|
| `yr` / `mn` / `ym` | INTEGER/TEXT | 年份、月份、年月 |
| `c1` | INTEGER | 全卡種合計：發卡總數 |
| `c2` | INTEGER | 全卡種合計：流通卡總數 |
| `c3` | INTEGER | 全卡種合計：當月交易筆數 |
| `c4` | INTEGER | 全卡種合計：當月交易金額（元） |
| `機構名稱` | TEXT | 申報機構名稱 |
| `資料月份` | TEXT | 資料月份（中文） |
| **各卡種數值欄（_N__ 格式，N = 0–17）** | | |
| `_N__card` | TEXT | 第 N 種卡 × 發卡總數（張） |
| `_N__card2` | TEXT | 第 N 種卡 × 流通卡總數（張） |
| `_N__txn_qty` | TEXT | 第 N 種卡 × 當月交易筆數（次） |
| `_N__txn_amt` | TEXT | 第 N 種卡 × 當月交易金額（元） |
| `_N__remark` | TEXT | 第 N 種卡 × 附註說明 |
| **具名彙總欄（系統標記欄，儲存選代表值或標籤）** | | |
| `儲值卡類型` | TEXT | 數值維度標籤（如「發卡總數」）— 系統標記欄 |
| `記名` | TEXT | 記名子類標籤（如「普通卡」）— 系統標記欄 |
| `聯名卡` | TEXT | 聯名卡子類標籤（如「與金融機構合作」）— 系統標記欄 |
| `與政府機關合作` | TEXT | 聯名卡×與政府機關合作×發卡總數（數值，= `_2__card`） |
| `與電信業者合作` | TEXT | 聯名卡×與電信業者合作×發卡總數（數值，= `_3__card`） |
| `與教育部認可之學校合作` | TEXT | 聯名卡×與學校合作×發卡總數（數值，= `_4__card`） |
